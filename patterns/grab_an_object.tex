\newcommand{\name}{Grab An Object}

% -------------------------------
% WAS

\newcommand{\desc}{Ein Datenobjekt auf einem Sender-Gerät soll von einem Empfänger-Gerät angefordert werden. Das Datenobjekt soll auch auf dem Empfänger-Gerät verfügbar sein.}

\newcommand{\solution}{Durch Auflegen eines Gerätes A auf ein Gerät B wird ein ausgewähltes Datenobjekt von B auf A übertragen. Nach dem Wiederwegnehmen ist das Datenobjekt auch auf A verfügbar.}

\newcommand{\useraction}{Der Anwender legt ein Gerät A auf ein Gerät B.

\newcommand{\reaction}{Das ausgewählte Datenobjekt von Gerät B wird auf dem Bildschirm von Gerät A angezeigt. }

\newcommand{\designnotes}{
\begin{itemize}
\item[-] Die Geräte müssen über einen Bildschirm verfügen, um die Daten darzustellen.
\item[-] Die Geräte müssen über definierte Schnittstellen miteinander kommunizieren, um das Datenobjekt zu senden und zu empfangen.
\item[-] Für eine intuitive Durchführung der Interaktion muss das Gerät, das die Daten bereitstellt, größer sein als das Gerät, das aufgelegt wird und das Datenobjekt mitnimmt/aufsaugt.
\end{itemize}
}
